VGrowConstraints

interface VGrowConstraints : Constraints

Marks a layout manager with growable children's height. It uses vpriority name since vgrow would clash with javafx.scene.layout.GridPane.setHgrow and javafx.scene.layout.GridPane.getHgrow.
