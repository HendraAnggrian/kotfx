KtfxBorderPane()

BorderPane with dynamic-layout dsl support. Invoking dsl will only set its content. There is currently no way to configure other areas (top, left, right, bottom) with dsl. Instead, create an instance and set it manually (e.g: left = ktfx.layouts.label()).
