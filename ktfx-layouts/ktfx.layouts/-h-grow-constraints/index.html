HGrowConstraints

interface HGrowConstraints : Constraints

Marks a layout manager with growable children's width. It uses hpriority name since hgrow would clash with javafx.scene.layout.GridPane.setHgrow and javafx.scene.layout.GridPane.getHgrow.
